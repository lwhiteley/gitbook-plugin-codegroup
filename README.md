GitBook CodeGroup Plugin
==============

### Template

<pre>
<code>
{% codegroup %}
    {% codetab "sdk" %}
        ```js
        var s = console1;
        ```
    {% codetab %}
        ```js
        var s = console2;
        ```
{% endcodegroup %}
</code>
</pre>

The `codetab` block accepts one argument. This is a custom tab name. if not specified, then the language of the code is used.

Pull requests are welcome
